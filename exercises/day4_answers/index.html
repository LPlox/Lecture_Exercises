Design systems

A design system is a collection of reusable components that can be used to build web applications and websites. Design systems help teams create consistent, high-quality user interfaces quickly by providing reusable components for developers to use in their projects. They also make it easier for designers to collaborate with developers on creating new features because they have clear documentation about how each component should behave and look. You can think of a design system as the "recipe book" for making web applications or websites in your company or team.
